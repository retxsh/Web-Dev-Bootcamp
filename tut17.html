Font Properties in CSS

We have learned about font and its properties in CSS. A few of then are mentioned below.

Properties:

  • CSS font-family: we use the font-family property to specify the font of a text.

  • CSS font-style: The font-style property specifies the font style for a text.

  • CSS font-size: The font-size property sets the size of a font.

  • CSS font-weight: The font-weight property sets how thick or thin characters in text should be displayed.

  • CSS line-height : The line-height property specifies the height of a line.
