Margin and Padding

Today I learned about margin, padding, border and box model in the CSS layout.
Padding is the distance between the content and the border. The fun part is you can give padding of same value to every side (top, right, bottom and left) or you can give it base on the axis i.e. x-axis(left and right) and y-axis(top and bottom) or you you can give different value to each and every side individually.

Now as we know the ways to adjust padding, let's move on to work with the margin. Margin in simple words is used to create space arounf elements, outside the border. But what about the ways to adjust the margins?
If you know how to adjust padding, your problem is solved because you have to use the same commands, just replace padding with margin in the stylesheet command.

I hope adjusting padding and margin is all clear by. Now, there is one issue, when we specify the width in stylesheet and if we adjust the padding then it's widht also changes. If you don't want this to happen, you can avoid it by using, box-sizing: border-box;
