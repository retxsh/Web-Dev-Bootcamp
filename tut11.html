HTML Entities

- Browser might mix reserved character with tags.
- Character entities are used to display reserved character in HTML. Like <(gt), >(lt), etc.

A few character entities are as follows:

  • Non-Breaking space -     (Two space is printed her)
  • Less than - <
  • Greater than - >
  • Ampersand - &
  • Pound - £
  • Copyright - ©
  • Empty character - ​ (Empty character is printed her)
For entities construct I used dev.w3.org for reference.
