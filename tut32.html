Attribute & nth Child Element Selectors

Let's talk about Attribute & nth Child Element Selectors. Attribute Selectors are used to select element with a specific attribute. For example, suppose there's a form having multiple input tags of various attributes like type, email, date etc. Now, if you want to style <input> tag have type="text", specifically.

Command: input[type='type'] {
/CSS Code Block/ }

nth Child Element Selectors are used to match every element i.e. the nth child, regardless of type of its parent. n: a number, a keyword or a formula. For example, suppose there's an unordered list with 10 list items. Now, we can target nth child in many ways like, li:nth-child(3) /will target third element in the list, li:nth-child(3n+3) /will target every third element of the list/, li:nth-child(even) /will target list items at even place/, etc.

Command: li:nth-child(number) {
/CSS Code Block/ }
